# Patternlab School

## Config

**Check Node**
`node -v`
If Node missing -> https://nodejs.org/en/

**Check Npm**
`npm -v`
If Npm missing -> https://nodejs.org/en/

**Check Gulp**
`Gulp -v`
If Gulp missing -> https://gulpjs.com/

## installation

1. Open the shell
2. Go to the folder location `cd`
3. `npm install` or `npm i`
4. `gulp patternlab:serve`
