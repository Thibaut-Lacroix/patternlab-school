# Patternlab School

## Config

**Check Node**
`node -v`
If Node missing -> https://nodejs.org/en/

**Check Npm**
`npm -v`
If Npm missing -> https://nodejs.org/en/

**Check Gulp**
`Gulp -v`
If Gulp missing -> https://gulpjs.com/

## installation

**Install**
`npm install` or `npm i`

**Build**
`gulp patternlab:build`

**Watch tasks**
`gulp patternlab:serve`
